# 📚 Expanded Literature v1.0 / Literatura Expandida v1.0

**Converse with classic literature characters using RAG technology**  
**Conversa con personajes de literatura clásica usando tecnología RAG**

---

## 🎯 What It Does / ¿Qué hace?

- Allows users to chat with characters from classic literature (currently Poe & Kafka)  
- Uses RAG (Retrieval-Augmented Generation) for context-aware responses  
- Generates first-person responses matching each character's personality  

---

- Permite conversar con personajes de literatura clásica (actualmente Poe & Kafka)  
- Usa RAG para respuestas contextualizadas  
- Genera respuestas en primera persona con el estilo del personaje  

---

## 📖 Demo Characters / Personajes de demostración:

- **The Tell-Tale Heart Narrator** (*Edgar Allan Poe*)  
- **The Black Cat Narrator** (*Edgar Allan Poe*)  
- **Gregor Samsa** (*The Metamorphosis - Franz Kafka*)  

---

## ⚠️ Important Notice / Aviso Importante:

This publication serves as a **demo/sample** of a personal literature interaction system. Full implementation requires original book files due to copyright restrictions. For private/home usage inquiries, contact the author after obtaining legal copies.

---

Esta publicación sirve como **demo/muestra** de un sistema personal de interacción con libros. La implementación completa requiere archivos originales de libros por restricciones de derechos de autor. Para consultas de uso privado/domiciliario, contacta al autor tras obtener copias legales de los libros deseados.

---

## 🛠️ Tech Stack:

- **Models**: Qwen2.5-1.5B-Instruct (multilingual)  
- **Embeddings**: paraphrase-multilingual-MiniLM-L12-v2  
- **Framework**: Transformers + SentenceTransformers + Gradio  
- **Language**: Spanish/English support
